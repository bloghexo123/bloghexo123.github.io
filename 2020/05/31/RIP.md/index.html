RIP

老梁 · 2020-5-31 · 次阅读

解析pc1Ip地址172.18.1.1/28：

IP地址：172.18.1.1
子网掩码：255.255.240.0
        1111 1111.1111 1111.1111 1111.1111 0000
        相与
        172.18.1.0000 0001
        也就是讲1111 0000
               0000 0001开始相与
得到0000 0000
网络地址为172.18.1.0也就是网段号

但由于题意知道：
网关地址要求该网段最后一位有效地址

由于网络号为28位，所以也就是最后4位，也就是讲当其

172.18.1.0000 1111主机号全为1，是广播地址

172.18.1.0000 1110也就是最后一位，算出来为14

所以网关地址为14

以上pc2，pc3均试用

解析R1,R2,R3:

172.16.123.0/30 因为子网掩码为30位
172.16.123.0000 0000 
172.16.123.0000 0001
172.16.123.0000 0010
172.168.123.0000 0011为广播地址
所以只能一个端口为172.16.123.1
   还有一个端口为172.16.123.2

R1配置：

[Huawei]system-view 
[Huawei]sysname R1
[R1]interface GigabitEthernet 0/0/1
[R1-GigabitEthernet0/0/1]ip address 172.18.1.14 28
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/0 
[R1-GigabitEthernet0/0/0]ip address 172.16.123.1 30
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/1
[R1-GigabitEthernet0/0/0]ip address 172.16.123.5 30
[R1]rip
[R1-rip-1]version 2 
[R1-rip-1]undo summary 
[R1-rip-1]network 172.18.0.0 
[R1-rip-1]network 172.16.0.0

R2配置：

[Huawei]system-view 
[Huawei]sysname R1
[R1]interface GigabitEthernet 0/0/1
[R1-GigabitEthernet0/0/1]ip address 172.18.2.14 28
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/0 
[R1-GigabitEthernet0/0/0]ip address 172.16.123.2 30
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/1
[R1-GigabitEthernet0/0/0]ip address 172.16.123.9 30
[R1]rip
[R1-rip-1]version 2 
[R1-rip-1]undo summary 
[R1-rip-1]network 172.18.0.0 
[R1-rip-1]network 172.16.0.0

R3配置：

[Huawei]system-view 
[Huawei]sysname R1
[R1]interface GigabitEthernet 0/0/1
[R1-GigabitEthernet0/0/1]ip address 172.18.3.14 28
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/1 
[R1-GigabitEthernet0/0/0]ip address 172.16.123.1030
[R1-GigabitEthernet0/0/1] interface GigabitEthernet 0/0/2
[R1-GigabitEthernet0/0/0]ip address 172.16.123.6 30
[R1]rip
[R1-rip-1]version 2 
[R1-rip-1]undo summary 
[R1-rip-1]network 172.18.0.0 
[R1-rip-1]network 172.16.0.0
